## Blog
после клонировании проекта
- **composer install**
- **cp .env.example .env**
- **php artisan key:generate**
- **проект создавался на базе sqlite**
- **создать файл базы данных, подключить ее в проект**
- **php artisan migrate**


Тренировочный проект проект реализован на основе laravel 8.0
composer create-project --prefer-dist laravel/laravel project-name "8.*.*"
